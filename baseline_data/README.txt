Baseline data are the data collected from each subject at their initial sampling.

baseline_microbiomedata
===================
'baseline_otu_phylum.txt' contains microbiome data at the Phylum Level for the baseline samples.
'baseline_otu_class.txt' contains microbiome data at the Class Level for the baseline samples.
'baseline_otu_genus.txt' contains microbiome data at the Genus Level for the baseline samples.
'baseline_otu.txt' contains otu table for the baseline samples.


baseline_surveydata
===================
'baseline_meta.txt' represents survey data containing essential variables (such as ID, age, gender, and others) for the baseline samples. 
'baseline_survey.txt' represents survey data for the baseline samples. 



baseline_sampleID.txt
===================
'baseline_sampleID.txt' contains the sampling IDs for the baseline data.
