baseline_microbiomedata
===================
'baseline_otu_phylum.txt' contains microbiome data at the Phylum Level for the baseline samples. 13016 samples 65 taxa 
'baseline_otu_class.txt' contains microbiome data at the Class Level for the baseline samples.   13016 samples 205 taxa 
'baseline_otu_genus.txt' contains microbiome data at the Genus Level for the baseline samples.   13016 samples 2037 taxa 
'baseline_otu.txt' contains otu table for the baseline samples.
